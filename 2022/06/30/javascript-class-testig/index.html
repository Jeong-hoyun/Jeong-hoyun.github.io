javascript 클래스 사용법 예시1

실무에서 개발을 할때 (react,vue) 물건과 같은 오브젝트화가 필요한 경우가 많다
또한 동일한 종류의 객체를 여러 개 생성해야 하는 경우도 필요하다

이럴 때 new 연산자와 생성자 함수에서 배운 new function을 사용할 수 있습니다.

클래스가 하는일

class User {…} 문법 구조가 진짜 하는 일은 다음과 같습니다.

User라는 이름을 가진 함수를 만듭니다. 함수 본문은 생성자 메서드 constructor에서 가져옵니다. 생성자 메서드가 없으면 본문이 비워진 채로 함수가 만들어집니다.
getFullName 같은 클래스 내에서 정의한 메서드를 User.prototype에 저장합니다.
new User를 호출해 객체를 만들고, 객체의 메서드를 호출하면 함수의 prototype 프로퍼티에서 설명한 것처럼 메서드를 prototype 프로퍼티를 통해 가져옵니다. 이 과정이 있기 때문에 객체에서 클래스 메서드에 접근할 수 있습니다.

지금까지 했던 설명을 코드로 표현하면 밑에 부분과 같습니다

class User {
  constructor(name) { this.name = name; }
    getFullName(){
        return   `${this.firstName} ${this.lastName} `
    }
}

// 클래스는 함수입니다.
alert(typeof User); // function

// 정확히는 생성자 메서드와 동일합니다.
alert(User === User.prototype.constructor); // true

// 클래스 내부에서 정의한 메서드는 User.prototype에 저장됩니다.
alert(User.prototype.sayHi); // alert(this.name);

// 현재 프로토타입에는 메서드가 두 개입니다.
alert(Object.getOwnPropertyNames(User.prototype)); // constructor, getFullName
